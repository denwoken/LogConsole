# LogConsole
 класс LogConsoleWidget предоставляет удомный виджет для вывода и сортировки логов.
 Виджет поддерживает:
 - сортировку по уровню логирования Debug, Warning, Info...
 - Включение/отключение отображение Даты/Времени/уровня логов/функции источника.
 - Смену цветовой палитры Даты/Времени/уровня логов/функции источника/ сообщения.
 - Сортировку по функциям источникам. Например, возможно отключить логи из функции main.
 - Изменение шрифта

 Примерная скорость работы:  
 - Загрузка 10000 строк ~ 5сек  
 - Сортировка в заисимости от параметров < 5сек
